# coco-extender
Extension cards for Radio Shack Color Computers

Directory poor-yaks-mpi-2a is a simple PCB with 4 or 5 edgecard
sockets.  It does no buffering nor switching of signals,
so it is more like a "Y Cable" than like a real MPI.
But I find it useful!   See the README.md in that directory.

Directory coco-extender-v1/ is the older CoCoExBo by Rachel Keslensky.  It has
one Edgecard Female and one 40-pin header suitable for a 40-wire ribbon
cable, in addition to the edgecard male that inserts into the CoCo slot.
(Probably you should use the poor-yaks-mpi-2a/ PCB above, instead.)
